# Tienda de Camisetas - FrontEnd Store

Proyecto frontend de una tienda de camisetas. Incluye catálogo de camisetas, descripción sobre la tienda y permite visualizar productos individuales, ver precios y seleccionar opciones (talla y cantidad).

**Demo:** https://nqingstore.netlify.app/

## Tecnologías

- HTML5
- CSS3 (Grid, Flexbox)
- Responsive Design

## Características

- Catálogo productos.
- Página individual para cada producto con formulario de compra (selección de talla y cantidad).
- Hover en productos para efecto de zoom (transition).
- Sección “Nosotros”.
- Bloques de ventajas de compra con iconos.
- Diseño responsive para móvil, tablet y desktop.
